# BraillePrinter with Cloud Speech-to-Text API
2019년 한이음 공모전에 출품할 프로젝트인 Cloud Speech-to-Text API를 이용한 점자프린터 입니다.

# Overview
음성 인식을 이용하여 점자로 변환하는 프린터입니다.

음성 인식 및 합성, 텍스트 처리는 라즈베리파이가 담당하고

점자 변환, 모터 제어는 STM32F4가 담당합니다.
